# Sassを利用する

sassをインストールする。

```
npm i sass
```

## 変数やmixinなど共通で利用するスタイルをグローバルに定義する

例)src/styles/variables.scssを作成

`astro.config.mjs`に設定を追加する。
ちなみに、`as *`は名前空間の指定で、仮に`as var`とした場合は`var.$color-primary`といった指定になる。

```
export default defineConfig({
  ---中略---
  vite: {
    css: {
      preprocessorOptions: {
        scss: {
          additionalData: `@use "src/styles/variables.scss" as *;`
        }
      }
    }
  }
});

```

# リセットCSSを追加

リセット用のCSSには[the-new-css-reset](https://elad2412.github.io/the-new-css-reset/)を利用します。

```
npm i the-new-css-reset
```

全てのファイルに利用する`src/Layout.astro`などで`the-new-css-reset`をimportします。

```
---
import "the-new-css-reset";
~~~中略~~~
---
```
