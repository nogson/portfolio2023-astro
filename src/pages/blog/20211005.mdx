## テストのリストを表示

```injectablephp
$ vendor/bin/phpunit --list-tests
```

## テストを実行する

```injectablephp
$ vendor/bin/phpunit
```

## 特定のファイルのテストを実行する

```injectablephp
$ vendor/bin/phpunit -v tests/Feature/myTest.php
```

## 特定のファイルの特定のテストを実行する

```injectablephp
$ vendor/bin/phpunit -v --filter myFuncTest tests/Feature/myTest
```
